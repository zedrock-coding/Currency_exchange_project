# Currency Converter

This is a simple currency converter application that fetches real-time exchange rates from an API and allows you to convert between different currencies.

## How to Run

1.  **Clone the repository or download the files.**
2.  **Open the `index.html` file in your web browser.**

That's it! The application will run locally in your browser without any additional setup.

## How it Works

The application is built with HTML, CSS, and JavaScript.

*   **`index.html`**: Contains the structure of the web page, including the input fields, dropdowns, and buttons.
*   **`style.css`**: Contains the styles for the web page.
*   **`script.js`**: Contains the logic for the application, including:
    *   Fetching currency data from the [ExchangeRate-API](https://www.exchangerate-api.com/).
    *   Populating the currency dropdowns.
    *   Performing the currency conversion.
    *   Handling loading and error states.

## Dependencies

There are no external dependencies required to run this application. It uses the browser's built-in `fetch` API to get data from the currency API.
